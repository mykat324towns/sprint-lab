# Exercise Scoring Workflow

## Objective
Build a small, high-value accessory pool for Phase 0 max velocity development.

## Steps
1. Generate candidate exercises
2. Run Exercise Scorer on each
3. Reject low-transfer or high-fatigue options
4. Keep only 4–6 maximum
5. Store final verdicts in Exercise Library

## Output
A ranked accessory pool with keep / maybe later / reject tags.
