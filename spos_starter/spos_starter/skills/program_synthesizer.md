# Program Synthesizer

## Objective
Turn athlete context, scored exercises, constraints, and current limiter hypothesis into a usable training structure.

## Rules
- Keep the phase goal pure
- Do not mix incompatible objectives
- Sprint work drives the phase
- Lifting supports sprinting, not the other way around
- Prefer simpler structures when transfer is equal
- Preserve the core KPI: repeatable, clean, non-forced fast reps

## Output
- Weekly structure
- Sprint session design
- Lift integration
- Progression rules
- Hold/regress criteria
