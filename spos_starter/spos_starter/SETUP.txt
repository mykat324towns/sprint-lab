SPOS Starter Setup

1. Copy .env.example to .env
2. Add your API keys
3. Install dependencies:
   pip install -r requirements.txt
4. Install Playwright browsers:
   playwright install
5. Start with:
   - skills/exercise_scorer.md
   - workflows/exercise_scoring_workflow.md

First useful prompt:
'Generate 12-15 Phase 0 accessory exercises for max velocity development, then score them using the Exercise Scorer.'
